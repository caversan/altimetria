## Listas de Exercícios

### [Lista 1 - Processamento de Dados Gradeados](./lista-1/)

Desenvolvimento de habilidades no processamento e análise de dados gradeados multi-satélite. Os alunos trabalham com dados de ADT (absolute dynamic topography), SWH (significant wave height) e VEGEO (geostrophic velocity) para uma região oceânica definida. O exercício envolve a geração de mapas mensais com análise de variações espaciais e temporais, estudo de séries temporais através de métodos estatísticos e espectrais, além da elaboração de diagramas espaço-temporais que caracterizam a dinâmica oceânica da região selecionada.
